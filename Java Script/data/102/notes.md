# JavaScript Data 102
## Three simple data types

### string
A string is defined with `'` or `"`
```
let location = `Atlanta, Georgia`;
```

### boolean
The value of a boolean is either `true` or `false`.
```
const isNew = false;
``` 

### number
pretty straightforward but `number` cannout write really, really big numbers.
```
let age = 32;
```
